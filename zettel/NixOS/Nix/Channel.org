#+title: Channel

A channel is a name for the latest *verified* git commits in [[file:Nixpkgs.org][Nixpkgs]].
Each channel has a different definition of what *verified* means (thinking about unstable).

We can say a channel is a version of [[file:Nixpkgs.org][Nixpkgs]].
