Le LISP est simplement un langage basé sur l'évaluation de listes.

** Conseils

Pour déclarer une liste, il faut un single-quote en préfixe, comme ci-dessous.

#+begin_src common-lisp
'(hello world)
#+end_src

Car dans ce langage, préfixer avec ='= indique à l'interpréteur de ne pas évaluer le code qui suit.
